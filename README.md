# Boundary Docs

The official documentation site for [Boundary](https://www.npmjs.com/package/boundary-sdk), a production-grade API integration SDK.

## Overview

Boundary standardizes third-party API interactions with unified typing and predictable response shapes. This documentation site provides comprehensive guides, API references, and best practices for integrating Boundary into your applications.

## Key Features

- **Unified API**: Standardized interface for multiple providers.
- **Typed Responses**: Predictable, type-safe return values.
- **Error Handling**: Consistent error shapes across all integrations.
- **Industrial Design**: A "Monochrome Luxury" aesthetic for a premium developer experience.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Raghaverma/Boundary-Docs.git
    cd boundary-site
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    pnpm dev
    ```

4.  **Open locally:**
    Visit [http://localhost:3000](http://localhost:3000) to view the documentation.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animation**: GSAP
- **Syntax Highlighting**: Shiki
